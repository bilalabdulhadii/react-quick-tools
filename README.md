## Installation & Setup

1. **Clone the repository**

    ```bash
    git clone https://github.com/bilalabdulhadii/react-quick-tools.git
    cd react-quick-tools
    ```

2. **Install dependencies**

    ```bash
    npm install
    ```

3. **Start the development server**

    ```bash
    npm start
    ```

    The app runs at `http://localhost:3000`.

4. **Build for production**

    ```bash
    npm run build
    ```

## Usage Guide

1. **Browse tools** from the grouped list or menu.
2. **Open a tool** to access its dedicated inputs and outputs.
3. **Export results** (e.g., QR/barcode PNG or SVG where supported).
